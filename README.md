# SafeLearn – AI Powered Disaster Preparedness Platform

SafeLearn is an AI-powered web platform designed to educate students, teachers, and communities about disaster preparedness through interactive learning modules, virtual drills, and an intelligent AI assistant.

The platform helps users understand how to respond to emergencies such as fires, floods, earthquakes, and other disasters in a safe and engaging way.

## Live Demo
https://safelearn-kappa.vercel.app/

## Features

- AI-powered chatbot for disaster preparedness guidance
- Interactive learning curriculum
- Virtual disaster simulation drills
- AI risk assessment based on location
- Leaderboards and progress tracking
- Multi-language support
- Responsive modern UI
- Real-time user interaction
